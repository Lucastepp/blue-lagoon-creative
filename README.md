# Blue Lagoon Creative

Concept website for a hospitality-focused creative service offering photography, film,
drone footage, websites and marketing for resorts, hotels, holiday rentals and
destination properties.

## Local Development

```bash
npm install
npm start
```

Open `http://localhost:4200/`.

## Validation

```bash
npm test -- --watch=false
npm run build
```

## Production Checklist

- Replace demonstration photography with approved client or commissioned imagery.
- Replace video placement previews with confirmed YouTube/Vimeo reels and accessible embeds.
- Connect the project form to a secured delivery path.
- Confirm business contact details, services and portfolio claims.
- Set the production domain, canonical/OG image, sitemap and indexable `robots.txt`.
- Remove the `noindex` metadata only after the above items are complete.
